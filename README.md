# 项目概述

本项目基于淘宝用户行为日志数据，对用户行为路径、转化效率及用户价值结构进行系统分析，主要包括：

- 用户行为结构分析（浏览、加购、收藏、购买）
- 转化率与漏斗模型分析
- RFM 用户分层建模
- 商品转化率及分布结构分析
- 时间趋势与增长率分析

项目完整体现从数据处理到可视化表达的分析闭环。

数据来源于阿里云天池公开数据集：

https://tianchi.aliyun.com/dataset/649

- 原始数据规模较大（约3GB，亿级行为记录），未上传至本仓库
- 仓库中仅保留清洗后与分析相关的结果数据文件

# 架构设计

本项目采用分层式分析架构：

## 1. 数据处理层（Python）

- 原始行为数据清洗
- 时间字段转换与维度拆分
- 转化率计算
- 用户 RFM 分层模型构建
- 中间结果数据表生成

## 2. 数据结构层（MySQL）

- 结果数据结构化存储
- 分组统计与指标复现
- 表粒度设计与结构规范

## 3. 可视化建模层（Power BI）

- 指标度量建模（DAX）
- 用户分层可视化
- 时间趋势分析
- 分布结构展示
- 多页面结构化报表设计

整体架构体现分析流程的分层思想：

数据处理 → 数据结构化 → 指标建模 → 可视化表达
